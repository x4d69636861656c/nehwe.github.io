The problem

A local firm's WordPress site took over seven seconds to load, ranked poorly, and had accumulated a dozen plugins over the years — several with known vulnerabilities that had never been patched.

The solution

I rebuilt the site from scratch on Hugo: hand-coded, no plugin dependencies, optimized images, and clean semantic markup with proper structured data for local search.

Beyond the speed and security wins, the client’s team can now edit content directly in Markdown without touching a page builder — which means fewer accidental layout breaks and no more waiting on a developer for a simple text change.
